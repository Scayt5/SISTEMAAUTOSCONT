📖 Guía de Entorno y Despliegue Técnico

Sigue estas instrucciones secuenciales para inicializar el backend de Inteligencia Artificial y la base de datos distribuida en cualquier entorno local que cuente con Visual Studio Code.

1. Prerrequisitos de Software
Docker Desktop

Debe estar abierto y en ejecución para gestionar el clúster de PostgreSQL.

Python 3.11 / 3.12 / 3.13

Requerido para interpretar la lógica y las redes neuronales.

2. Extensiones Mandatorias en VS Code

Instala estas herramientas desde el Marketplace de extensiones de VS Code (Ctrl+Shift+X):

Python (Microsoft) Habilita el autocompletado y análisis de sintaxis de Python.
Live Server (Ritwick Dey) Permite renderizar y refrescar este archivo HTML de forma automática.
3. Secuencia de Comandos en Consola

Abre una terminal integrada en VS Code (Ctrl+`) en la raíz del proyecto y ejecuta en orden:

# Paso A: Levantar e inicializar la bóveda de datos aislada
docker compose up -d

# Paso B: Construir el entorno virtual de aislamiento
python -m venv venv

# Paso C: Activar el entorno virtual (Elige según tu terminal)
# Si usas Windows PowerShell: .\venv\Scripts\Activate.ps1
# Si usas Windows Git Bash / Linux: source venv/bin/activate

# Paso D: Instalar ráfaga de dependencias y modelos (YOLO, EasyOCR, FastAPI)
pip install -r requirements.txt

# Paso E: Lanzar el servidor web ASGI en tiempo real
uvicorn main:app --reload
Nota de Ejecución: Al procesar la primera imagen, el sistema descargará de forma automatizada los pesos de la red neuronal convolucional de YOLOv8 Nano (yolov8n.pt). Por favor, asegúrate de contar con una conexión a internet activa durante la primera prueba.
